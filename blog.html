Question-1: What are the different ways to select an element in the DOM?

We can select an element in the dom using getElementById,getElementsByClassName,querySelector,getElementsByTagName,querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML:- it reads both the HTML markup and the text content of the element. innerText:- it returns the text as it appears on screen. It ignores HTML tags textContent:- it returns only the text. Whiles innerText reads text as it is rendered on screen

Question-3: What is event delegation in the DOM?

Event delegation is a JavaScript technique where an event listener is attached to a parent element instead of attaching event listeners to multiple child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a type of DOM event propagation where the event first triggers on the innermost target element and then successively triggers on the ancestors (parents) of the target element in the same nesting hierarchy till it reaches the outermost DOM element or document object

Question-5: How do you create, add, and remove elements using JavaScript?

We can create, add, and remove elements using JavaScript like document.createElement('div'); document.body.appendChild(newElement); newElement.remove();
